🚗 We Ride

We Ride is a ride-sharing application designed to connect riders with nearby drivers quickly and safely. Mo
The goal of this project is to build a modern mobility platform similar to services like  and .

---

📌 Features

- Rider account creation
- Driver registration
- Request a ride
- Driver matching
- Ride tracking
- Payment processing
- Rating and reviews

---

🧠 Tech Stack

Frontend

- React / React Native

Backend

- Node.js
- Express.js

Database

- PostgreSQL

Development

- GitHub Codespaces
- Visual Studio Code

---

⚙️ Installation

Clone the repository:

git clone https://github.com/Serakwana977/We-Ride.git

Navigate into the project:

cd We-Ride

Install dependencies:

npm install

Run the development server:

npm start

---

🗄️ Database Setup

Run the database schema:

psql -f schema.sql

This will create the required tables for:

- Users
- Drivers
- Vehicles
- Rides
- Payments
- Reviews
